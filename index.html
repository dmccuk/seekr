FAQ

How do I set up a game?
Choose a game zone, share a code with friends, pick roles, then start the countdown. Everything runs in the browser.
Who can see who during the game?
Seekers see fugitives as last known pings with an uncertainty circle, for example 50 meters. Fugitives never see seeker locations. The map shows last known pings, not exact points.
When is the extraction location revealed?
The host presses Set extraction point, the game picks a hidden spot inside the zone. The exact location is revealed to fugitives when 20 percent of game time remains. Seekers are notified that extraction has been disclosed, they do not see where it is.
How do fugitives win?
Reach the extraction point before time runs out. You only need to get within 10 meters of the extraction point.
How do seekers tag fugitives?
The game registers a tag when players are within the set tag distance, for example 5 to 10 meters. Hosts choose the tag distance on the setup page.
What is the default ping interval for seekers?
Five minutes by default. Hosts can change the interval on the setup page to be more often or less often.
What are Tokens and Actions?
Every player starts with 500 Tokens. Actions cost 25 Tokens each. Examples, Decoy Ping shows a short lived fake location, Delay Reveal postpones your next fugitive ping, Instant Reveal is a seeker only pulse that requests the next fugitive ping sooner. No pay to win.
Can the host see the extraction location?
No. The location is hidden from everyone until the 20 percent reveal, then fugitives see it. Seekers only see that it has been revealed.
What game modes are supported?
Many seekers versus one or two fugitives, team versus team across a larger zone, large zone hunts with a late extraction reveal. Everyone can start anywhere inside the zone.
Do I need an app?
No app needed. Visit the site on iPhone or Android, enter the code, and play. Location updates work while the page is open and visible. For long games we will offer a native app with background location.
How accurate is GPS?
Outdoors, accuracy is usually within a few meters. In dense or indoor areas it can drift. Open spaces work best. The game uses uncertainty circles to keep things private.
Is it free to play?
Yes. Core gameplay is free. Actions use Tokens, there are no pay to win advantages.
Is it safe and private?
Play in safe areas, respect property and local laws, stay aware of your surroundings. The game shows blurred locations by design, for example a 50 meter uncertainty, not exact points.
What if someone loses signal?
The game uses the latest known position. If a device drops offline for a while, seekers will keep seeing the last ping with its uncertainty. When the device reconnects, pings resume.
Can we set our own rules?
Yes. Hosts can set duration, ping interval, tag distance, zone size, and which Actions are allowed. Extraction reveal time is fixed at 20 percent in the demo.
